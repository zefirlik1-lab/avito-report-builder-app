# Анализ правок среднего приоритета (REPORT_DIFF_ANALYSIS)

## 1. Поле «Номер на Авито»

### Сравнение с expected_report.xlsx
- В **expected_report.xlsx** в колонке «Номер на Авито» на листе «Админ панель» подставлены **телефонные номера** (например, «8 931 009-42-68»).
- В **output_structure.md** зафиксировано: «Номер на Авито в примере выглядит как телефонный номер, а не ID объявления».

### Проверка входного файла (input_example.xlsx, input_structure.md)
- Во входном файле **нет колонки с контактным телефоном** (ни «Телефон», ни «Контактный телефон», ни «Номер телефона»).
- Есть колонки:
  - **«Номер объявления»** → маппится в `ad_id` (в примере — ID вида 7837068432 из HYPERLINK);
  - **«Посмотрели телефон»** → метрика `phone_views` (число просмотров телефона), а не сам номер.

### Вывод
**Телефон во входном файле отсутствует.** Подставлять вымышленные номера нельзя. В колонке «Номер на Авито» продолжаем выводить **ID объявления** (`avito_number_display` = значение из «Номер объявления»), пока в структуре входа не появится поле с контактным телефоном. Логика зафиксирована в коде (комментарий в `admin_sheet.py`).

## 2. Лист «Диаграмма»

### Число строк
- **output_report.xlsx:** 161 строка (1 шапка + 160 данных) — по числу объявлений во входном файле.
- **expected_report.xlsx:** 176 строк (1 шапка + 175 данных) — видимо, сформирован из другого набора данных (другой файл или период).

Разница **160 vs 175** объясняется разным источником данных, а не ошибкой: мы строим лист «Диаграмма» из того же датасета, что и «Админ панель» (одна строка на объявление). Искусственно добавлять строки не требуется.

### Структура и наполнение
- Колонки совпадают: Кабинет Авито, Тип, Название, Адрес, ID объявления.
- Данные заполняются из того же `ReportDataset`, что и для остальных листов.

### Автофильтр
- В **expected_report.xlsx** на «Диаграмма» включён автофильтр с диапазоном A1:E161 (или по факту до последней строки).
- В нашем pipeline автофильтр для «Диаграмма» уже задаётся в **formatter**: по `workbook_layout.yaml` у листа «Диаграмма» стоит `autofilter: true`, и при форматировании выставляется диапазон **A1:E&lt;max_row&gt;** (фактически A1:E161 при 160 строках данных). Дополнительных правок не требуется.
